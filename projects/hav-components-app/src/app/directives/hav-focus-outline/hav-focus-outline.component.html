## FocusOutlineDirective

Description

This directive is used to add an outline to the host element when it receives focus, and to reset the outline to its previous value (or else 'none') when it loses focus. The host receives focus when the focusin event fires, and loses focus when the focusout event fires.

FocusOutlineDirective is included in the component library as a utility to facilitate focus indication in other applications, but it is also used internally in the library. All hav-components that are interactable should already be both keyboard accessible and use visual focus indicators (to comply with WCAG standards). Some of them use this directive, some use other visually similar techniques.

Tab through or click the examples to see the directive in action.

Default
Custom focus outline
Outline offset inside host element.
